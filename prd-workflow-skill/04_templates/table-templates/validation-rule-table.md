# 校验规则表

> Belongs to: `05_context/writing-standards/`  
> Source: PRD 需求自查清单 V3.3 · Sheet 07  
> Version: v1.0.0

适用场景：必填校验、格式校验、重复校验、提交前校验、审批前校验。

必须回答：什么时候校验，按什么条件校验，失败时是否阻断，提示什么。

## 空白模板

```
校验编号 | 校验对象 | 校验时机 | 校验条件 | 通过标准 | 失败提示 | 是否阻断 | 校验层级 | 备注
```

| 列名 | 含义 |
|------|------|
| 校验编号 | 唯一标识，如 V001 |
| 校验对象 | 被校验的字段或操作 |
| 校验时机 | 什么时候触发校验（如"点击保存时""提交审批时""失焦时"） |
| 校验条件 | 校验的判定条件 |
| 通过标准 | 数据达到什么状态算通过 |
| 失败提示 | 校验不通过时展示给用户的文案 |
| 是否阻断 | 是 / 否 — 校验失败是否阻止操作继续 |
| 校验层级 | 前端 / 后端 / 前端+后端 |
| 备注 | 关联规则编号、特殊处理说明 |

## 示例

| 校验编号 | 校验对象 | 校验时机 | 校验条件 | 通过标准 | 失败提示 | 是否阻断 | 校验层级 | 备注 |
|---------|---------|---------|---------|---------|---------|---------|---------|------|
| V001 | 模板名称 | 点击保存时 | 模板名称为空 | 模板名称已填写 | 请输入模板名称 | 是 | 前端+后端 | 保存时必须二次校验 |
| V002 | 模板名称 | 点击保存时 | 同业务模块下名称重复 | 当前名称未重复 | 模板名称已存在，请修改后保存 | 是 | 后端 | 按所属模块+模板名称查重 |
| V003 | 消息正文 | 点击保存时 | 正文超过 500 字 | 正文不超过 500 字 | 消息正文不能超过 500 字 | 是 | 前端 | 字数含中文、英文、标点 |

## 不合格写法

| 问题写法 | 为什么不合格 |
|---------|-------------|
| 名称不能为空 | 没写校验时机、失败提示、是否阻断 |
| 名称不可重复 | 没写查重范围和组合条件 |
