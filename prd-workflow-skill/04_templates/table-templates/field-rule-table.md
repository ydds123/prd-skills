# 字段规则表

> Belongs to: `05_context/writing-standards/`  
> Source: PRD 需求自查清单 V3.3 · Sheet 07  
> Version: v1.0.0

适用场景：表单字段、列表字段、详情字段、配置字段。

必须回答：字段是什么，谁填写，怎么填写，从哪里来，能不能改。

## 空白模板

```
字段名称 | 字段含义 | 字段类型 | 输入/展示 | 是否必填 | 长度/格式 | 默认值 | 选项来源 | 字段状态 | 空值展示 | 备注
```

| 列名 | 含义 |
|------|------|
| 字段名称 | 字段在页面上的标签名称 |
| 字段含义 | 这个字段的业务含义 |
| 字段类型 | 文本 / 开关 / 时间 / 下拉单选 / 下拉多选 / 多选列表 等 |
| 输入/展示 | 用户输入 还是 系统展示（只读） |
| 是否必填 | 是 / 否 |
| 长度/格式 | 字符长度限制或格式约束 |
| 默认值 | 系统预设值 |
| 选项来源 | 下拉选项从哪里来（"系统固定枚举"/"用户手动输入"/引用其他数据） |
| 字段状态 | 可编辑 / 只读 / 禁用 / 自动生成 |
| 空值展示 | 没有数据时显示什么 |
| 备注 | 校验规则编号、查重规则、联动规则等 |

## 示例

| 字段名称 | 字段含义 | 字段类型 | 输入/展示 | 是否必填 | 长度/格式 | 默认值 | 选项来源 | 字段状态 | 空值展示 | 备注 |
|---------|---------|---------|---------|---------|---------|--------|---------|---------|---------|------|
| 模板名称 | 消息模板名称 | 文本 | 输入 | 是 | 1-32 字符 | 无 | 用户手动输入 | 可编辑 | 不适用 | 同一业务模块下不可重复 |
| 启用状态 | 模板是否启用 | 开关 | 输入 | 是 | 启用 / 停用 | 启用 | 系统固定枚举 | 可编辑 | 不适用 | 停用后不触发消息 |
| 最近更新时间 | 最近一次保存时间 | 时间 | 展示 | 否 | YYYY-MM-DD HH:mm | 系统生成 | 系统记录 | 只读 | -- | 保存成功后自动更新 |

## 不合格写法

| 问题写法 | 为什么不合格 |
|---------|-------------|
| 填写名称 | 没写长度、是否必填、是否重复、是否可编辑 |
| 展示更新时间 | 没写时间格式、更新时间触发条件、空值展示 |
