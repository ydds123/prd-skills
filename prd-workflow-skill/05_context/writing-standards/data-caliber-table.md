# 数据口径表

> Belongs to: `05_context/writing-standards/`  
> Source: PRD 需求自查清单 V3.3 · Sheet 07  
> Version: v1.0.0

适用场景：列表数据来源、报表数据来源、看板指标、统计口径。

必须回答：数据从哪里来，怎么算，按什么范围算，多久更新，谁能看。

关键提醒：数据类需求必须写数据为空、数据延迟、数据口径不一致的处理方式。

## 空白模板

```
指标/数据项 | 业务定义 | 数据来源 | 计算口径 | 统计范围 | 更新时间 | 单位/精度 | 权限过滤 | 空值/异常值处理 | 备注
```

| 列名 | 含义 |
|------|------|
| 指标/数据项 | 指标或数据项名称 |
| 业务定义 | 这个指标代表什么业务含义 |
| 数据来源 | 数据从哪个系统/表/接口来 |
| 计算口径 | 计算公式或聚合逻辑 |
| 统计范围 | 时间范围、组织范围、筛选条件 |
| 更新时间 | 实时 / T+1 / 每小时 / 每日 |
| 单位/精度 | 条/百分比/元，保留几位小数 |
| 权限过滤 | 不同角色看到的数据范围是否不同 |
| 空值/异常值处理 | 分母为0、数据延迟、异常值时的展示和兜底 |
| 备注 | 与其他指标的关联、特殊说明 |

## 示例

| 指标/数据项 | 业务定义 | 数据来源 | 计算口径 | 统计范围 | 更新时间 | 单位/精度 | 权限过滤 | 空值/异常值处理 | 备注 |
|-----------|---------|---------|---------|---------|---------|---------|---------|---------------|------|
| 未读消息数 | 当前用户未读站内信数量 | 站内信消息数据 | 接收人=当前用户 且 状态=未读 | 当前登录用户 | 实时 | 条，整数 | 按用户过滤 | 无数据展示 0 | 不包含已删除消息 |
| 审批待办数 | 当前用户待处理审批数量 | 审批任务数据 | 当前节点审批人=当前用户 且 状态=待审批 | 当前登录用户 | 实时 | 条，整数 | 按审批权限过滤 | 无数据展示 0 | 已超时待办仍计入 |
| 异常关闭率 | 已关闭异常数占异常总数比例 | 巡检异常记录 | 已关闭异常数 / 异常总数 x 100% | 默认近 30 天 | T+1 | 百分比，保留 1 位小数 | 按部门权限过滤 | 分母为 0 时展示 -- | 需说明异常总数是否含已作废 |
